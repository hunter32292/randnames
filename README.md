# randnames
generate random names

## Example:
```
package main

import (
  "log"
  
  "github.com/hunter32292/randnames"
)

func main(){
  log.Println(randnames.GiveName())
}
```

## Output:
```
2020/09/09 10:25:13 Marney
```
